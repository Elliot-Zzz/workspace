一、Tailscale 是什么？

Tailscale 是一种基于 WireGuard 的虚拟组网工具，它能够：

  • 在不同设备之间建立安全的 点对点连接
  • 通过内网 IP 直接访问目标设备，无需公网 IP
  • 无需端口映射或配置防火墙规则
  • 开箱即用，支持多平台（Windows、macOS、Linux、iOS、Android）

二、Tailscale Ubuntu 安装教程

1. 服务器端安装（Ubuntu/Debian）

# 添加 Tailscale 软件源
curl -fsSL https://pkgs.tailscale.com/stable/ubuntu/noble.noarmor.gpg | sudo tee /usr/share/keyrings/tailscale-archive-keyring.gpg >/dev/null
curl -fsSL https://pkgs.tailscale.com/stable/ubuntu/noble.tailscale-keyring.list | sudo tee /etc/apt/sources.list.d/tailscale.list

# 安装 Tailscale
sudo apt-get update
sudo apt-get install tailscale

# 启动并连接（服务器建议禁用密钥过期）
sudo tailscale up --operator=root --accept-routes --advertise-exit-node

2. 客户端安装

  • Windows/macOS：从 官网 下载安装包
  • Linux：同上，使用相同的安装命令
  • iOS/Android：应用商店搜索 "Tailscale"

3. 登录与认证

sudo tailscale up

执行后会显示一个 URL，浏览器打开并使用 Google/微软账号登录即可。

4. 查看设备 IP

# 查看 Tailscale 分配的 IP（IPv4）
tailscale ip -4

# 查看所有设备
tailscale status

5. 禁用密钥过期（服务器推荐）

sudo tailscale cert --expiry-safe 87600h

或者在管理后台设置：Tailscale Admin Panel

方案详解

1. Tailscale（推荐 ⭐⭐⭐⭐⭐）

  • 优点：端到端加密、无需公网 IP、无需配置防火墙、组网简单
  • 缺点：需要安装客户端、初次需要认证
  • 免费版：支持 3 个用户、无限设备

2. 传统 VPN（OpenVPN/IPSec）

  • 优点：企业级安全、功能全面
  • 缺点：配置复杂、需要公网 IP 或中继服务器

3. SSH 密钥 + 关闭密码登录

  • 优点：最安全、无需额外软件
  • 缺点：只能 SSH 访问，不支持其他服务

4. IP 白名单

  • 优点：简单有效
  • 缺点：IP 变化时需要手动更新（可结合 DDNS 解决）

5. FRP 内网穿透

  • 优点：无需公网 IP、配置简单
  • 缺点：依赖中继服务器、有流量限制、安全性取决于服务端

五、安全建议

💡 基于本次被入侵事件的教训：
  1. 禁用 root 的 SSH 密码登录，只允许密钥认证
  2. 使用 Tailscale 替代直接暴露公网端口
  3. 定期更换密码（即使使用密钥）
  4. 启用防火墙，只开放必要端口
  5. 监控登录日志，发现异常及时处理

六、总结

对于个人用户和小型团队，Tailscale 是目前最佳的选择。它兼顾了安全性和便捷性，无需复杂的配置即可实现安全的远程访问。

如果仅需要 SSH 访问，SSH 密钥 + Tailscale 的组合是最安全的方案。
