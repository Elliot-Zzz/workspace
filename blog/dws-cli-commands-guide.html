钉钉 DWS CLI 命令实战总结

已掌握 12 个命令 · 待探索 8 个功能 · 实战日期：2026-04-30

一、已掌握的命令 ✅

1. 通讯录 — contact

dws contact user get-self --format json
获取当前用户信息（userId、所属部门等）
dws contact dept search --query "研发" --format json
按部门名称关键词搜索部门，返回部门ID
dws contact dept list-members --ids 40182869 --format json
查询指定部门的所有成员（姓名 + userId）
dws contact user search --query "姓名" --format json
按姓名关键词搜索用户，返回 userId 列表
dws contact user search-mobile --mobile 13800138000 --format json
通过手机号查 userId

2. 考勤 — attendance

dws attendance record get --user <USER_ID> --date 2026-04-30 --format json
查询指定用户某日的考勤记录（打卡时间、排班状态等）
dws attendance shift list --users userId1,userId2 --start 2026-04-27 --end 2026-05-03 --format json
批量查询团队排班（最多50人，间隔不超过7天）
dws attendance rules --date 2026-04-30 --format json
查看考勤规则（打卡范围、弹性时间等）

3. 待办 — todo

dws todo task create --title "任务标题" --executors <USER_ID> --priority 30 --format json
创建待办任务（priority: 10低/20普通/30较高/40紧急）
dws todo task list --page 1 --size 20 --status false --format json
查询未完成（false）或已完成（true）的待办列表
dws todo task done --task-id <ID> --status true --format json
标记指定待办为已完成
dws todo task delete --task-id <ID> --yes --format json
删除待办（⚠️ 不可逆，需 --yes 确认）

4. 日历 — calendar

dws calendar event create --title "标题" --start "2026-05-06T10:00:00+08:00" --end "2026-05-06T10:30:00+08:00" --format json
创建日程，自动发送钉钉提醒（默认提前15分钟）

5. 邮箱 — mail

dws mail mailbox list --format json
查看当前邮箱账号列表
dws mail message search --email <邮箱地址> --query "subject:*" --size 5 --format json
搜索邮件（KQL 语法，支持 date:2026-04-30 等条件）
dws mail message get --mail-id <ID> --format json
查看邮件完整内容
dws mail message send --to "xxx@xxx.com" --subject "标题" --content "正文" --format json
发送邮件

6. 群聊消息 — chat

dws chat message send --group <群ID> --title "标题" --text "内容" --at-all --format json
以当前用户身份发群消息，支持 @所有人
dws chat message send-by-webhook --token <WEBHOOK_TOKEN> --title "标题" --text "内容" --format json
通过自定义机器人 Webhook 发群消息（无需认证，适合告警场景）
dws chat message search --keyword "关键词" --start "2026-04-01T00:00:00+08:00" --end "2026-04-30T23:59:59+08:00" --format json
按关键词搜索群聊消息记录
dws chat message list-mentions --start "2026-04-30T00:00:00+08:00" --end "2026-04-30T23:59:59+08:00" --format json
拉取某时间段内 @我的所有消息
dws chat search --query "群名关键词" --format json
搜索群会话列表

二、待探索功能 🔮

1. OA 审批 — oa

已验证可用的命令：

dws oa approval list-pending --format json
查看待我审批的单子（processInstanceId）
dws oa approval list-forms --cursor 0 --size 20 --format json
查看企业可用审批表单模板（processCode）
dws oa approval detail --instance-id <ID> --format json
查看审批单详情（表单字段、附件、当前状态）
dws oa approval approve --instance-id <ID> --task-id <ID> --remark "同意" --format json
同意审批（⚠️ 需先获取 taskId）
dws oa approval reject --instance-id <ID> --task-id <ID> --remark "拒绝理由" --format json
拒绝审批（⚠️ 需先获取 taskId）

2. AI 表格 — aitable

适合运维场景：工单管理、监控数据报表、资产台账。

dws aitable base list --format json
列出最近访问的 AI 表格
dws aitable base search --query "关键词" --format json
搜索 AI 表格
dws aitable base get --base-id <ID> --format json
获取表格详情（tableId、dashboardId 等）
dws aitable record list --base-id <ID> --table-id <ID> --format json
列出数据表中的所有记录
dws aitable dashboard get --base-id <ID> --dashboard-id <ID> --format json
查看图表仪表盘

3. 钉钉文档 — doc

适合运维场景：故障复盘文档、运维手册、知识库。

dws doc search --query "故障复盘" --format json
搜索钉钉文档
dws doc node get --node-id <ID> --format json
读取文档内容
dws doc node create --title "标题" --folder-id <ID> --format json
创建新文档
dws doc block create --node-id <ID> --block-type "text" --content "内容" --format json
向文档写入块级内容

4. 云盘 — drive

适合运维场景：日志文件归档、备份包存取、报告下载。

dws drive file list --space-id <ID> --format json
列出云盘文件
dws drive upload-info --file-name "xxx.log" --file-size <字节数> --format json
获取文件上传信息（上传URL）
dws drive commit --file-name "xxx.log" --file-size <字节数> --upload-id <ID> --format json
确认上传完成
dws drive download --file-id <ID> --format json
下载文件

5. AI 听记 — minutes

适合运维场景：值班交接会议自动生成纪要、提取待办事项。

dws minutes list --format json
获取听记列表
dws minutes get --minutes-id <ID> --format json
获取听记摘要、转写、待办
dws minutes mindmap --minutes-id <ID> --format json
获取思维导图
dws minutes speaker-list --minutes-id <ID> --format json
获取发言人列表

6. 日志/日报 — report

适合运维场景：团队日报收集、运维日志汇总统计。

dws report template list --format json
查看可用日志模板
dws report create --template-id <ID> --content "日报内容" --format json
按模板创建日报
dws report inbox --format json
查看收到的日志（收件箱）
dws report detail --report-id <ID> --format json
查看日志详情

7. DING 消息 — ding

适合运维场景：P0 故障紧急电话/短信通知。

dws ding message send --robot-code <CODE> --users <USER_ID> --type app --content "内容" --format json
发送应用内 DING（免费）
dws ding message send --robot-code <CODE> --users <USER_ID> --type call --content "紧急内容" --format json
发送电话 DING（⚠️ 有通信成本，紧急时使用）
dws ding message send --robot-code <CODE> --users <USER_ID> --type sms --content "内容" --format json
发送短信 DING（⚠️ 有通信成本）

8. 群组管理 — chat group

dws chat group create --name "群名" --users userId1,userId2 --format json
创建内部群
dws chat group members list --id <群ID> --format json
查看群成员列表
dws chat group members add --id <群ID> --users userId1,userId2 --format json
添加群成员
dws chat group rename --id <群ID> --name "新群名" --format json
修改群名称
dws chat group members add-bot --id <群ID> --robot-code <CODE> --format json
添加机器人到群

三、暂不支持的功能 ❌

  • 群公告（Announcement）：DWS CLI 目前没有群公告专用命令，普通群消息无法置顶，实用性大打折扣
  • 部门组织架构全量查询：根部门 ID=1 返回空，需从已知部门逐步展开，缺少全量树形结构接口
  • 通讯录批量导出：list-members 只能按部门查，无法一次性导出全公司成员
  • Corp Token 应用凭证登录：DWS 不支持 AppKey/AppSecret 直接登录，只能用 OAuth 设备流，每个新 Scope 首次需扫码授权

四、授权机制说明 🔐

DWS CLI 使用 OAuth 2.0 个人授权，refresh_token 有效期 30 天，每个新 API Scope 首次使用需要扫码授权一次。

常见报错：
  • PAT_MEDIUM_RISK_NO_PERMISSION → 该 Scope 尚未授权，命令行会返回授权链接，点击扫码即可
  • .AUTH_TOKEN_EXPIRED → Token 过期，执行 dws auth login --force 重新登录
  • InvalidParameter → 参数错误，检查 flag 名称（kebab-case）和格式
一次性搞定所有授权：把所有会用到的命令都跑一遍授权，之后 30 天内无需再弹框。
技巧：遇到授权弹框时，命令行会直接输出完整的授权 URL，复制给用户在浏览器打开即可完成授权，无需在服务器端操作。

五、命令发现技巧 🔧

遇到不确定的命令时，按以下优先级查阅：

  1. dws <cmd> --help — 人类可读的 Usage、Example、Flags
  2. dws schema <product>.<command> — 机器可读 Schema（参数名、必填、别名）
  3. dws schema — 列出所有产品和命令
dws schema calendar.event create --jq '.tool.flag_overlay'
查看 calendar.event create 的所有 flag 别名
dws schema calendar.event create --jq '.tool.required'
查看 calendar.event create 的必填字段
