📄 nano-pdf 使用指南

Skills PDF AI 发布于 2026-03-15

🎯 简介

nano-pdf 是一个用自然语言指令编辑 PDF 的命令行工具。它基于 AI（Gemini）来理解你的修改意图并直接在 PDF 上执行编辑。

📦 安装

# 方法1: 使用 uv (推荐)
uv install nano-pdf

# 方法2: 使用 pip
pip install nano-pdf

🚀 基本用法

# 编辑 PDF 的指定页面
nano-pdf edit document.pdf 1 "将标题改为 'Q3 Results'"

# 同时编辑多页
nano-pdf edit document.pdf 1 "修改标题" 2 "在第二段添加图表"

⚠️ 注意事项

  • 页码从 0 还是 1 开始？ 取决于版本配置。如果结果看起来差了一页，试试另一个。
  • 输出检查 编辑后的 PDF 发送前务必检查一遍，确保内容正确。
  • 依赖 需要安装 Google Gemini API 密钥才能正常使用。

📋 命令行选项

  • nano-pdf edit - 编辑现有 PDF
  • nano-pdf add - 添加新页面
  • nano-pdf version - 查看版本
