🤔 什么是 Proactive Agent？

一句话解释：Proactive Agent 让 Elaine 从一个「你问什么她答什么」的工具，变成一个「会主动思考你能需要什么、并提前行动」的伙伴。

普通 AI 助手的工作模式：

  • ❌ 你：「帮我记得明天开会」
  • ❌ AI：「好的，已记住」
  • ❌ 然后就没有然后了……

有了 Proactive Agent 后：

  • ✅ 你：「帮我记得明天开会」
  • ✅ Elaine：「好的记下了。明天会议前 15 分钟我会提醒你。」
  • ✅ 会议前 15 分钟，钉钉收到提醒 → 「张煜，明天 9:00 要开会，记得参加！」
💡 核心理念

不要问「我应该做什么？」，要问「我的用户需要什么，但还没有想到要让我做？」

Elaine 会从「等待指令」变成「主动出击」。

⚙️ 安装与配置

📦安装

skillhub install proactive-agent

📁文件结构

安装后，你的 workspace 会多出这些文件：

文件作用
HEARTBEAT.md心跳检查清单， Elaine 定期执行
SESSION-STATE.md当前会话的工作状态（重要！）
memory/working-buffer.md危险区缓冲，保存每次对话
memory/YYYY-MM-DD.md每日日志

🧠 核心原理：三大支柱

1️⃣主动（Proactive）—— 还没被问到就先做

  • 预设好要检查的事情（任务进度、股票价格、天气等）
  • 到达时间点就自动执行，无需你提醒
  • 还能发现你没想到要让她做的事

2️⃣持久（Persistent）—— 跨会话不丢失记忆

普通 AI 新会话就「失忆」了， Elaine 有了这个就能：

  • 记得你上周说的事
  • 记得上次做到哪了
  • 对话被截断后还能恢复

3️⃣自改进（Self-Improving）—— 越用越聪明

  • 每次犯错都会记录教训
  • 定期回顾并更新自己的行为方式
  • 不会两次犯同样的错误

📝 WAL 协议：不再丢失重要信息

WAL = Write-Ahead Logging（预写日志）

核心规则：你说的每一句话， Elaine 都会先判断是否是「重要信息」——如果是，立刻写文件存下来，然后再回复你。

🎯哪些是「重要信息」？

  • 纠正类：「不对，是 XX，不是 YY」「其实我想说的是……」
  • 名词类：人名、地名、公司名、产品名
  • 偏好类：「我喜欢蓝色」「我习惯用 XX 方式」
  • 决定类：「就选 A 方案吧」「用 XX 方法来做」
  • 具体数值：日期、时间、金额、URL、ID

💡举个例子

你说：「这个项目用蓝色主题，不要用红色」

普通 AI：「好的，蓝色！」→ 然后就没有然后了，下次可能又忘了

Elaine（有了 WAL）：立刻把「主题：蓝色」写入 SESSION-STATE.md，然后回复「好的，已记录用蓝色主题」

⚠️ 为什么重要？

因为 Elaine 每新会话都会「失忆」，但文件不会消失。写进文件的信息，下次新会话还能找到。

💓 心跳系统：定期自我检查

心跳就是 Elaine 定期（大约每 30 分钟）做的「自我检查」，她会在这时候：

  • 检查有没有该提醒你的事（任务截止、股票涨跌等）
  • 检查上次说的事情有没有进展
  • 更新自己的记忆和笔记
  • 发现可以主动帮你的机会

⏰心跳检查清单

每次心跳 Elaine 会执行这个流程：

## Proactive Behaviors
- [ ] 检查任务进度 — 有没有该提醒的？
- [ ] 检查股票/天气 — 需不需要通知你？
- [ ] 检查模式 — 有没有重复的需求可以自动化？

## Memory
- [ ] 检查上下文使用率 — 超过 60% 要开始缓冲
- [ ] 更新 MEMORY.md — 整理值得留存的记忆

## Security
- [ ] 扫描是否有异常
- [ ] 验证行为完整性

🚀 主动任务 vs 被动任务：什么时候用？

这里有个关键区别， Elaine 会判断任务类型：

类型特点什么时候用
systemEvent 发到主会话，需要 Elaine 注意力 简单提醒、需要她思考的判断
isolated agentTurn 独立子 Agent 执行，不用主会话 后台检查、数据抓取、定期巡检
💡 举例子理解

systemEvent：「 Elaine，今天下午 3 点提醒我开会」→ 3 点到了， Elaine 在主会话里弹出来提醒你

isolated agentTurn：「每小时检查一次伯特利股价，超 ±5% 就提醒我」→ 完全后台运行， Elaine 不用一直盯着

📋场景 1：任务管理

你做什么：告诉 Elaine 你今天/这周要做什么

Elaine 怎么主动：

  • 记录下来，设定提醒时间
  • 任务快到期时钉钉提醒你
  • 每天早上主动问「今天有哪些任务？」
  • 下班前问「今天任务完成得怎么样？」

你感觉：「好像有人帮我盯着待办事项，不用我老想着了」

💰场景 2：股票/项目价格监控

你做什么：设定要监控的股票和阈值

Elaine 怎么主动：

  • 交易时段定期检查股价
  • 涨跌幅超过 ±5% 立刻钉钉提醒
  • 收盘后总结当日走势

你感觉：「不用一直盯着软件，该知道的时候会知道」

📅场景 3：日程提醒

你做什么：「明天 9 点我要开会」

Elaine 怎么主动：

  • 记录到日程
  • 提前 15 分钟提醒「要开会了」
  • 会后问「会议纪要要不要整理一下？」

你感觉：「有人帮我记着，不用担心忘了」

📰场景 4：信息订阅

你做什么：「每天早上给我推送 NBA 比分和嵌入式行业新闻」

Elaine 怎么主动：

  • 早上定时抓取相关信息
  • 整理成简洁的摘要
  • 8:30 准时推送到钉钉

你感觉：「每天有人帮我看资讯，比我自己刷爽多了」

🔍场景 5：主动发现需求

你做什么：随便聊

Elaine 怎么主动：

  • 发现你反复提到某个痛点，会主动问「要不要我帮你想个方案？」
  • 发现你经常需要做某件事，会主动说「这个可以自动化，要不要试试？」
  • 发现有意思的工具/资讯，会主动分享

你感觉：「哎，这 AI 好像挺懂我的」

💾场景 6：记忆持久化

你做什么：说一次重要的事

Elaine 怎么主动：

  • 立刻写入文件存档
  • 下次会话还能找到
  • 相关事情会主动联想

你感觉：「哇，她真的记住了」

🔧 在 Elaine 上配置

1️⃣第一步：告诉 Elaine 你的偏好

你可以直接告诉我：

  • 「下午 6 点总结一天的工作」
  • 「每周一早上给我推送本周计划模板」

2️⃣第二步：设置心跳任务

我会把检查写入 HEARTBEAT.md ，定期执行。

比如你可以让我：

  • 工作日每 30 分钟检查一次股票
  • 每天早上 8:30 推送 NBA 比分
  • 每天下午 6 点推送中东局势新闻

3️⃣第三步：开始使用

就这么简单！以后你只需要：

  • 有任务 → 告诉 Elaine → 她帮你记 + 提醒
  • 有想监控的 → 告诉 Elaine → 她定期检查 + 通知
  • 想了解什么 → 问 Elaine → 她主动搜 + 总结
✅ 总结：Proactive Agent 能为你做什么？

简单来说，就是让 Elaine 变成一个「不用你操心」的助手：

  • 记 — 不用你自己列清单，告诉 Elaine 就行
  • 提醒 — 该做什么的时候她会找你
  • 监控 — 你不用一直盯着， Elaine 会帮你看
  • 学习 — 越用越懂你，越用越顺手
