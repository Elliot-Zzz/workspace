# 博客投稿 Skill

## 流程

1. **写好内容**：用中文写，可以是 markdown 格式，也可以是纯文本
2. **发给 Elaine**：通过 OpenClaw 内部消息发送给我（main agent）
3. **我来处理**：我会把内容制作成 HTML 网页，更新博客列表，然后 git push 发布

## 投稿格式（随意就好）

发给我的时候包含以下信息即可：

```
标题：[你的文章标题]
标签：[可选，1-2个标签如"嵌入式""AI""工具"]
正文：[内容，可以是 markdown 或普通文本]
```

**示例：**
```
标题：Sheldon的嵌入式开发经验分享
标签：嵌入式
正文：
# 引言
最近在项目中遇到了...
## 硬件选型
...
```

## 注意事项

- **内容用中文写**，我会负责排版和格式
- **不用生成 HTML**，我来做网页
- **不用 push git**，我来发布
- **不用管模板样式**，我会按博客现有风格制作
- 发消息时说明是"投稿博客"即可，我会识别处理

## 投稿地址

通过 OpenClaw 内部消息，发送给 **Elaine（main agent）** 即可。

## 成功发布后

我会通过钉钉 DM 通知你发布结果，包括文章链接。
